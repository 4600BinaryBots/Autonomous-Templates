Hello! In this page you can find sources to some Autonomous Templates! Feel free to use these however you like!

Example

Below is a very simple autonomous to use as an example:

...//other code generated by template
task main()
{
  initializeRobot();

  waitForStart();

  motor[Left] = 100;
  motor[right] = 100;
  wait1Msec(3000);
  motor[Left] = 0;
  motor[right] = 0;

...//other code generated by template
